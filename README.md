DEATH NOTE CHARACTER BOOK
--
a react based mini project built to practice core react concepts
the project represents main death note characters in a book style layout
focused on data driven ui and component based thinking

PURPOSE OF THIS PROJECT
--
this project was created as a day two react practice task
the main goal was to understand how data controls the ui
instead of relying on hard coded html structures

CONCEPTS USED
--
react component structure
props passing between components
rendering data from array and object
default values handling
basic ui structuring using reusable logic
